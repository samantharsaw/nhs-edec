There are 3 tags in the service:

  • Not started
  • Started
  • Completed

Examples of the 3 tags we use

Pages the tags appear on

Tags appear on two pages in the service to:

  • show progress overall for the practice's submission
  • show progress per sections on the task list

When tags should be used

On the select a practice page

Not started - should appear if a user has not saved any answers they've changed/given in any of the sections or confirmed any of the sections as complete.

Started - should appear when a user has made changes to answers/answered questions and saved them or confirmed completion of at least one section.

Completed - should appear if a user has confirmed all sections as complete and submitted their electronic declaration.

On the task list page

Not started - should appear if a user has not saved any answers they've changed/given in the section or confirmed the section as complete.

Started - should appear when a user has made changes to answers/answered questions and saved them.

Completed - should appear if a user has confirmed the section as complete.
