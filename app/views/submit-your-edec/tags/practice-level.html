How many practice level tags there are

There are 4 practice level tags

  • Not started
  • Started
  • Ready to submit
  • Submitted

Where practice level tags appear

Practice level tags appear on the select a practice page and the practice page to show progress for a practice declaration submission.

Practice level tags appear on the:

  • Select a practice page (shows all 4 tags)
  • Your practice page

When practice level tags should be used

Not started - should appear if a user has not started any sections or saved any answers they've changed/given in any of the sections.

Started - should appear if a user has made changes to answers/answered questions within a least one section for the practice and saved their progress.

Ready to submit - should appear if a user has confirmed all sections as complete but has not yet submitted their declaration.

Submitted - should appear if a user has submitted their declaration.
