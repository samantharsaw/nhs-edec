How many section level tags there are

There are 3 section level tags

  • Not started
  • Started
  • Completed

Where section level tags appear

Section level tags appear on the task list to show progress per sections on the task list.

Task list showing the 3 tags

When section level tags should be used

Not started - should appear if a user has not started the section or saved any answers they've changed/given in the section.

Started - should appear if a user has made changes to answers/answered questions within the section and saved them.

Completed - should appear if a user has answered/checked all questions within the section and confirmed the section as complete.
